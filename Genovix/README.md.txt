# Genovix: Pharma Intelligence Dashboard

Genovix is a pharma intelligence dashboard that helps generic drug manufacturers identify FDA-approved formulations using interactive visualizations and machine learning.

## Key Features
- Interactive dashboard for pharma insights
- ETL pipeline integrating FDA datasets
- ML-based drug approval forecasting

## Tech Stack
- Python (Data processing, ETL, ML)
- SQL (Data integration)
- Tableau (Data visualization)
- HTML/CSS (Webpage interface)

## Folder Structure
- `Final_Report/`: Final project report
- `Scripts/`: Python scripts for data processing and analysis
- `Tableau/`: Tableau workbook for visualization
- `Webpage/`: Genovix interactive webpage (`.html`)

> **Note:** Source data files are not included due to their size. They can be downloaded from the official [FDA website](https://www.fda.gov/).

## How to Use
1. Clone or download this repository.
2. Open the Tableau workbook in the `Tableau/` folder to explore the dashboard.
3. Open the `.html` file in the `Webpage/` folder with any browser to view the interactive webpage.
4. Run Python scripts from the `Scripts/` folder to process and analyze data.

## About
This project was developed as part of advanced analytics coursework in an academic setting with collaborative input.
